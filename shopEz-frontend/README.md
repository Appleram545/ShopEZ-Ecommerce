# ShopEZ - Angular E-Commerce Application

A full-stack e-commerce SPA built with Angular 17 (front end) and ASP.NET Core (back end).




## Setup & Run

# Install dependencies
npm install

# Start  server
ng serve

# Open browser

# http://localhost:4200


## Features

### Customer
- Browse products with category/search filters
- View product details
- Add items to cart (requires login)
- Checkout and place orders

### Admin
- Add / Edit / Delete products
- View all customer orders
